[Tap](https://github.com/bluesky-social/indigo/blob/main/cmd/tap/README.md) is an in-beta tool made by BlueSky that runs on localhost and provides a websocket.
It manages a [Firehose](firehose.md) connection with verification, backfill and filtering capabilities.
