# Exploring Lexicons

As a developer, you might want to explore already existing lexicons for many reasons. You might want to find out if there are already lexicons that you can reuse, or you might want to find out how other people have modeled their lexicons.

You have a couple tools you can use to explore lexicons:

- [lexicon.garden](https://lexicon.garden)
- [ufos.microcosm.blue](https://ufos.microcosm.blue)
