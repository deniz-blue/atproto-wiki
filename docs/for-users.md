# AT Protocol for Users

Applications that use the AT Protocol allow you to own your data and have control over where it is stored.

Traditional applications store your data on their servers, which means you have to trust them to keep it safe and private. With the **AT Protocol**, your data is stored on a [Personal Data Server (PDS)](./pds/index.md) that you choose. You can choose a PDS that you trust, or even run your own PDS if you want.

Any application built on top of the **AT Protocol** will work with any other application built on top of the **AT Protocol**. This means you can switch between applications without losing your data, and you can even use multiple applications at the same time if you want.
